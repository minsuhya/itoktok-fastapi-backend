# 20251025 수정사항

1. 초파 로그인 시 초파에 해당되는 프로그램, 상담사 목록 출력, 로그인 사용자 기준 데이터 출력 처리
   - 회원등급이 센터(user_type:1) 인 경우 상담사(user_type:2)의 데이터로 함께 조회.
   - 상담사(user_type:2)인 경우 본인의 관련 데이터만 조회 가능
2. 로그인 시 디폴트로 주간 일정페이지로 이동
3. Frontend의 주간일정에서 일정을 드래그앤드롭으로 13시에 12시로 옮겼으나 일정 상세 우측 슬라이드 페이지에는 그대로 13시로 남아있음
   - 화면에서는 변경 완료
4. 데일리 일정에 모든 일정 출력 됨 ==> 로그인 사용자 기준으로 처리 필요(1번 요구사항 참조)
5. 주간일정에서 일정 등록을 위한 화면을 클릭해서 신규 일정 등록 시 일정반복타입의 기본 선택값은 "매주"
   - 클릭한 요일이 상담일정 등록폼의 반복요일에 적용되고 시간도 기본 선택되도록 한다.
6. 우측 사이드바의 상담사 리스트 목록에서 상담사 선택 시
   - 현재 페이지가 WeeklyView.vue, MonthlyView.vue 일 경우 페이지 유지 후 데이터 갱신
   - 그 외 페이지일 경우 WeeklyView.vue(주간일정)으로 이동
7. 좌측 슬라이드 일정 상세페이지(ScheduleFormSliding.vue)에서 삭제 기능이 정상 작동하지 않은 오류 수정
8. 월간,주간,일간 일정페이지에서 일정의 상태가 "취소"일 경우, 일정에 취소선 처리
9. 일정 메모 등록/수정 확인
   - 신규 일정일 경우 반복 타입에 따른 모든 일정에 동일한 메모 등록
   - 일정 수정시는 (일정 수정 범위)에 따라 수정된 메모로 수정 필요
